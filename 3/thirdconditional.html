We use Third Conditional to...

Refer to hypothetical situations in the past, and how things would have turned out if something had happened or not.



Structure


If + Past Perfect + suj + would/wouldn't have + verb participle


We can replace "WOULD" with any of the other modal verbs, as long as they don't change, like "could, might", etc.

It is not necessary for "If" to go first, but neither "would" or "if" can be in the same sentence.



Affirmative structure


If + suj + had + verb participle + suj + would have + verb participle


If I had enought money, I would have bought to you a house.



Negative structure


If + suj + hadn't + verb participle + suj + wouldn't have + verb participle


If she hadn't fallen over, they wouldn't have looked at us.



Question structure


Would + suj + have + verb participle + if + suj + had + verb participle?


If it had rained, would you have had the picnic?
